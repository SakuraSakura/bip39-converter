# BIP39 Converter
Convert a 16.5-byte account seed to a 12-word phrase based on BIP39.

## Setting Up

### Prerequisites

- OpenSSL 1.1.1

### Building
```sh
git clone https://github.com/SakuraSakura/bip39-converter.git
cd bip39-converter
make
```
This will generate 2 executables `seed_conv` and `mytest`.
